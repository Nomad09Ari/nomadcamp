# NomadCamp

## 概要
キャンプ場のレビュー投稿サイトです。

## URL
https://nomadcamp.onrender.com/

## 使用技術
- Node.js
- Express
- MongoDB
- Cloudinary
- Bootstrap

## 主な機能
- ユーザー登録・ログイン
- 投稿機能（CRUD）
- 画像アップロード
- 地図表示

## 工夫した点（←超重要）
